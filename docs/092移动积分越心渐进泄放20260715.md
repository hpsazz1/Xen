# 移动积分越心渐进泄放

## 证据

r34 已将 left/right 自运动抑制降至约 6~7%，但 `FinalMx` 仍以约 3.3~3.7 Hz 翻转，`IntegralCountsX` 同步以约 2.5~3.4 Hz 翻转；reverse 仍约 2.36 Hz。预测门控不再是主要反馈源。

## 方案

生产链继续保留持续运动积分，避免 r33 完全解卷绕造成的 12 px 稳态偏差；当误差跨越稳定半径且积分仍指向旧侧时，对旧积分先乘 0.5 再叠加当前误差。这样保留低速追踪能力，同时限制换侧瞬间的反向脉冲。控制器修订号升至 r35，速度、响应、预测和积分窗口均不变。

## 验证

下一轮比较 `IntegralCountsX` 与 `FinalMx` 翻转 Hz、left/right 稳态误差和 reverse 平均输出。若翻转仍存在但不再与积分同步，则继续检查比例控制与设备量化；不再调整自运动门槛。
