# 物理响应扫描脚本

`tools/run_physical_response_probe_sweep.ps1` 按顺序运行孤立物理响应采集器，默认测试 `counts=16,32,64`，每组运行 3 次。每次写入 `OutputRoot\countsN_runM`，参数和 ROI 保持一致，便于比较位移线性和响应分位时间。

示例：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\run_physical_response_probe_sweep.ps1 `
  -ProbeExe D:\XenPhysicalProbe\xen_physical_response_probe.exe `
  -Config D:\XenPhysicalProbe\config.ini `
  -NdiSource "HPSAZZ (main)" `
  -OutputRoot D:\XenPhysicalProbe\sweep_20260719
```

可用 `-Runs 1` 只执行一轮，或用 `-Counts 16,32,64` 显式指定档位。

脚本遇到非零退出码默认保留当前目录并继续下一组，最终表格会列出每组退出码。需要在首个失败处停止时传入 `-StopOnFailure`。`-ContinueOnFailure` 保留用于兼容旧命令。不要在已有输出目录上重复运行。
