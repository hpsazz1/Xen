# 物理响应扫描脚本

`tools/run_physical_response_probe_sweep.ps1` 按顺序运行孤立物理响应采集器，默认测试 `counts=8,16,32,64`。每组写入 `OutputRoot\counts_N`，参数和 ROI 保持一致，便于比较位移线性和响应分位时间。

示例：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\run_physical_response_probe_sweep.ps1 `
  -ProbeExe D:\XenPhysicalProbe\xen_physical_response_probe.exe `
  -Config D:\XenPhysicalProbe\config.ini `
  -NdiSource "HPSAZZ (main)" `
  -OutputRoot D:\XenPhysicalProbe\sweep_20260719
```

脚本遇到非零退出码默认立即停止并保留当前目录；只有明确传入 `-ContinueOnFailure` 才会继续下一组。不要在已有输出目录上重复运行。
