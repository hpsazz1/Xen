# 静止预测阶段验收通过

## 背景与版本

2026-07-14使用 `BuildRevision=b7bde5e33681`、`ControllerRevision=14`、DML+NDI、3200 counts/s和预测50/50/1.0完成static九段复测。CSV身份与正式DML构建一致，全程未使用手动鼠标辅助。

## 采集状态

本轮NDI声明240 FPS，实际接收平均238 FPS、中位238 FPS、P95为240 FPS，已经恢复正常；DML推理平均96 FPS、中位95 FPS。上一轮约158.5 FPS的接收降速没有继续出现，因此本轮可以作为r14 static正式验收数据。

## 验收结果

X/Y预测生效率均为0.4%，达到不高于0.5%的验收要求。九段平均P95误差为X 4.16 px、Y 4.12 px；前八段预测生效率全部为0，只有trial 9出现两帧预测。

trial 9逐帧检查显示，两帧预测发生在约214~223 ms，随后约232 ms开始命中自运动门控并连续抑制至约311 ms。抑制期间目标稳定收敛到中心，保持结束后没有再次形成任何预测段。说明r14的8 px重新武装门槛已阻止越心回弹重新锁定，不存在r12/r13曾出现的门控结束后二次晃动。

本轮平均P95误差高于r13的2.26/3.57 px，但预测几乎完全关闭，误差变化来自不同起始位置、推理节奏和基础控制收敛，而不是预测提前量。继续修改预测门控无法改善这些非预测误差。

## 决策

static阶段正式通过并冻结r14算法。不得为了消除首次门控前仅有的两帧预测继续延长保持、提高普通首次预测门槛或削弱预测强度；这些修改会直接损害left/right和jump的真实移动响应，收益与风险不成比例。

本轮只更新分析文档和待办，不修改代码、配置、构建脚本或运行资源。按项目第16条执行纯非代码精简闭环：内容检查、`git diff --check`和中文本地提交；不重新构建、不推送云端，当前 `b7bde5e33681/r14` 测试版本继续有效。
