文件包含 819 个运行态观测和 1314 个暂停态观测。按 `AimPipelineOutputPaused` 的上升、下降沿切分出恰好 9 个运行段，各段运行帧数为 94、95、95、95、81、89、74、86、110。

九段运行态的 `AimPipelineManeuverModelActive` 全部为 0，运行态 settled 侵入为 0。全部 236 个机动启用帧均位于松键换位的暂停态，不计入 static 运行门禁；暂停态命令违例和观测连续性违例均为 0。因此 r63 九点 static 硬门槛正式通过，r62 第 5 点在响应结束后触发的反向过冲未再复现。

正式分析器的文件级结果为 PASS。仅隔离 static 文件运行时，overall 汇总会提示缺少 jump/reverse 运行态正样本覆盖；这是跨场景晋级条件，不是 static 文件失败，需由后续正样本实测补齐。
