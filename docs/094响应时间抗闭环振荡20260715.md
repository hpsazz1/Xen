# 响应时间抗闭环振荡

## 证据

r36 数据中中心迟滞没有降低翻转：left 输出约 6.07 Hz、right 约 4.24 Hz，积分翻转仍约 5.32/4.43 Hz。命令队列与设备应用一致，排除发送链路问题。80 ms 一阶响应对应的闭环带宽接近观测到的反复修正频率。

## 调整

将生产移动目标默认响应时间从 80 ms 调整为 120 ms，降低误差穿越后的比例修正带宽；3200 cps、500 ms 积分、预测 50/50/1.0 和中心反向迟滞保持不变。控制器修订号升至 r37。

## 验证

下一轮比较输出翻转 Hz、P95 误差、jump 丢失率和 reverse 恢复时间。若翻转下降且误差仍在可接受范围，保留 120 ms；若跟随明显变慢，再单独评估预测前瞻，不同时修改积分。
