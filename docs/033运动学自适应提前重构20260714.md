# 运动学自适应提前重构

## 背景

固定框外身位方案能够让预测点越过目标框，但目标框可能只包含头部或被遮挡的局部身体，框尺寸不能可靠代表目标距离。若再要求逐武器录入弹速、目标真实身高和弹道参数，会把预测模块变成不可验证的伪物理模型。因此本轮废弃目标框测距、固定框外身位和逐武器弹速假设。

## 设计

预测器只使用稳定可观测量：补偿程序自身视角运动后的目标速度、目标加速度、观测年龄和连续运动方向。提前距离按 `速度×前瞻时间 + 0.5×加速度×前瞻时间²` 计算，再乘统一的 `prediction_strength`。目标匀速时提前量稳定，持续加速时自动增加，减速时自动缩短；停止或疑似反向的第一帧立即撤销预测，连续两次反向后才切换方向。

该模型不声称计算真实弹着点。缺少可靠深度、命中反馈和武器弹道数据时，程序无法从单张二维检测可靠判断远近。相比伪造距离，运动学提前能够统一补偿检测延迟、控制响应和目标运动趋势，并保持参数最小闭环。

## 参数

保留启用开关、基础前瞻时间、速度平滑和预测强度四项。预测强度只缩放运动学结果，不改变方向确认、停止收回和丢失重置规则。目标框仍写入CSV供检测质量审计，但不参与预测公式。

## 视频验证

使用horizontal_jump、left、right、reverse和static五段2560×1440、240 FPS视频，按DML实测106 FPS发布率、10/15/20 ms观测年龄和2000 counts/s执行联合搜索。推荐50 ms前瞻、15 ms速度平滑、1.0强度：jump最差P95从关闭预测70.35 px降至16.73 px，左右均值从15.90 px降至8.45 px，反转从17.98 px降至8.13 px，静止X/Y保持1.26/5.58 px，所有场景均无裁剪丢失。

## 诊断与测试

流水线CSV新增 `PredictionStrength` 和 `PredictionAccelerationX/Y`，保留预测偏移、方向锁定和自身视角位移。移动分析器输出 `PredictionActivePct`、`P50PredictionLeadPx`、`P95PredictionLeadPx` 与 `PredictionSideFlipCount`。控制器修订号升级为7。单元测试覆盖匀速、加速、静止、停止、变向、异常跳变、目标丢失、自身视角时间线、配置迁移和UI入口。
