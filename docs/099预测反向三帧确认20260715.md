# 预测反向三帧确认（2026-07-15）

干净 r39 数据中，left/right 的预测未进入振荡抑制，但在误差约 `±3~15 px` 时会于 30~70 ms 内重新锁到相反方向，形成 `±1~4 counts` 可见脉冲。r39 的 reverse 专用 80 ms 响应分支没有进入 left/right，排除响应优化外溢。

本轮保持初次方向建立仍为两次稳健回归确认，只把“已有方向后的反转”提高到连续三次同向回归确认。真实持续 reverse 只增加约一个观测周期确认时间；单帧或双帧闭环回弹不再切换预测侧。未新增用户参数，也不修改响应、速度、积分和预测距离。

下一轮使用五类标准数据比较 left/right 换向频率、reverse 恢复时间和 jump P95，确认反转确认时间没有造成高速目标回退。
